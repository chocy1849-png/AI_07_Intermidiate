- 개선: 청크 단위 출처 정보를 expander로 펼쳐서 보여주기

```python
# 목표 출력 형태

for citation in result["citations"]:
    with st.expander(citation["file_name"]):
        st.write(f"청크: {citation['chunk_id']}")
        st.write(f"관련도: {citation.get('score', '-')}")
```
