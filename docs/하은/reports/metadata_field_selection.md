# Metadata Field Selection

## 목적

`data_list.csv`에서 챗 필터와 검색 보조에 바로 쓸 수 있는 핵심 메타데이터를 선정한다.

## 우선 사용 필드

- `발주 기관`: 사이드바 필터의 1순위 기준
- `사업명`: 검색과 결과 요약의 핵심 식별자
- `사업 금액`: 향후 규모별 필터와 분석용
- `공개 일자`: 최근 공고 정렬과 기간 필터용
- `입찰 참여 마감일`: 마감 임박/종료 여부 판단용
- `파일형식`: 파서 분기와 품질 비교용
- `파일명`: 원본 파일 매칭 키
- `source_path`: 실제 파싱 입력 경로

## 보조 필드

- `공고 번호`, `공고 차수`: 동일 공고의 차수 관리
- `사업 요약`: 검색 확장과 문서 카드 요약
- `csv_text`: 원본 파싱 전 임시 미리보기 및 비교 기준

## 제외 또는 후순위

- `입찰 참여 시작일`: 결측이 많아 1차 UI 필터 기준으로는 비권장

## 결론

1차 프로토타입에서는 `발주 기관`, `사업명`, `공개 일자`, `사업 금액`, `파일형식`, `파일명`을 핵심 메타데이터로 사용한다.
