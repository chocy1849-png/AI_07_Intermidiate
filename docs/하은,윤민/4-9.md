## Track B 오늘 할 일

### [B-5.1] 대화 세션 관리 고도화

**현재**: `st.session_state.messages` 하나. 새 대화 시작 불가.

**추가할 것:**
- 사이드바에 **"새 대화 시작"** 버튼 → `st.session_state.messages` 초기화
- **"대화 초기화"** 버튼 (같은 기능, 이름만 다르게 둬도 됨)
- 세션 목록은 `st.session_state.sessions` 리스트로 관리
  - 각 세션: `{"id": ..., "title": 첫 질문 20자, "messages": [...]}`
  - 사이드바에 세션 목록 버튼으로 나열 → 클릭 시 해당 세션으로 전환

```python
# 핵심 구조
if "sessions" not in st.session_state:
    st.session_state.sessions = []
if "current_session" not in st.session_state:
    st.session_state.current_session = {"id": 0, "messages": [...초기 메시지...]}

# 새 대화 시작
if st.sidebar.button("새 대화 시작"):
    # 현재 세션 저장
    st.session_state.sessions.append(st.session_state.current_session)
    # 새 세션 생성
    st.session_state.current_session = {"id": len(st.session_state.sessions), "messages": [...]}
    st.rerun()
```

### [B-5.3] 응답 스트리밍 구현

**현재**: `st.markdown(result["answer"])` 로 완성된 텍스트를 한 번에 출력.

**목표**: 답변이 타이핑되듯 스트리밍으로 출력.

방법 1 — `st.write_stream` (LangChain streaming callback 연동 시):
```python
with st.chat_message("assistant"):
    response = st.write_stream(rag_chain.stream(prompt))
```

방법 2 — 텍스트를 청크로 나눠 흉내내기 (LLM 스트리밍 미지원 시 임시):
```python
with st.chat_message("assistant"):
    placeholder = st.empty()
    displayed = ""
    for char in result["answer"]:
        displayed += char
        placeholder.markdown(displayed + "▌")
    placeholder.markdown(displayed)
```

> Track A의 RAG Chain이 streaming을 지원하면 방법 1,
> 아직이면 방법 2로 UI 흐름만 먼저 구현.

### [B-5.4] 검색 결과 디버그 패널

**현재**: `st.json(result["debug"])` — 숫자만 보임.

**목표**: 검색된 청크 내용을 expander로 펼쳐서 볼 수 있게.

```python
with st.expander(f"검색된 청크 {len(result['citations'])}건 보기"):
    for i, citation in enumerate(result["citations"]):
        st.markdown(f"**[{i+1}] {citation['file_name']}**")
        st.caption(f"chunk_id: {citation['chunk_id']} | score: {citation.get('score', '-')}")
        # 청크 텍스트가 있으면 표시
        if "text" in citation:
            st.text_area(
                label=f"chunk_{i}",
                value=citation["text"][:500],
                height=120,
                label_visibility="collapsed",
            )
        st.divider()

with st.expander("디버그 상세"):
    st.json(result["debug"])
```

> `result["citations"]` 에 `text` 키가 없으면 Track A에 요청.
> `rag_pipeline.py` 의 `generate_answer()` 반환값에 청크 텍스트 포함 여부 확인 필요.
