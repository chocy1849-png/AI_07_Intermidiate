■ 설계 원칙
  이 질문셋은 "표 enrichment 전후 성능 차이"를 측정하기 위해 설계됨.
  반드시 B-00 베이스라인에도 소급 평가하여 기준선을 확보해야 함.

  Group A (5개): PDF table blocks 이미 존재
    → 베이스라인도 일부 답변 가능
    → enrichment 전후 비교 기준선 역할
    → 정답 데이터 확보 용이

  Group B (5개): HWP <표> 마커만 존재, 본문에 내용 없음
    → 베이스라인은 답변 불가 또는 오답 예상
    → enrichment 후 정답률이 올라야 함
    → 표 enrichment 효과의 핵심 측정 대상

  Group C (5개): 표 내용 + 본문 텍스트를 함께 요구하는 복합 질문
    → 표만 있어도 안되고 본문과 결합해야 완전한 답변 가능
    → Completeness 지표 개선 여부 측정

  평가 지표 (기존 동일):
    Faithfulness / Completeness / Groundedness / Relevancy (1~5)
    judge: gpt-5

  소급 평가 필수:
    B-00 (표 enrichment 전 베이스라인)
    B-04 (표 enrichment 후 최종)
    → 두 결과 비교로 "표 enrichment의 실질적 기여도" 수치화

■ 평가 실행 지침
=============================================================

1. 소급 평가 (필수)
   B-00 베이스라인을 이 15개 질문으로 먼저 평가
   → Group B 5개에서 낮은 점수 확인 (enrichment 효과의 기준선)

2. enrichment 후 평가
   B-04 (표 enrichment 완료) 동일 15개 질문 평가

3. 비교 측정 포인트
   Group A : B-00과 B-04 점수 유사하면 정상 (이미 PDF 표 있음)
   Group B : B-04에서 유의미한 상승이 있어야 enrichment 효과 입증
   Group C : Completeness 지표 개선 여부가 핵심

4. judge: gpt-5 동일 사용
   채점 기준: Faithfulness / Completeness / Groundedness / Relevancy

5. 기대 결과 패턴
   Group A — B-00: 3.5~4.5 / B-04: 유사 또는 소폭 상승
   Group B — B-00: 1.0~2.5 / B-04: 3.0~4.5 (큰 폭 상승)
   Group C — B-00: 2.0~3.0 / B-04: 3.5~4.5 (중간 폭 상승)

   → Group B에서 큰 폭 상승이 확인되면
     "HWP 표 enrichment가 RAG 성능에 실질적 기여"를 수치로 증명 가능
